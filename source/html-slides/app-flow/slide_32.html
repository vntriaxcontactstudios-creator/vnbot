Memoria — Inferida y Contradicción

compare_arrows SUGERIR · MOSTRAR · DECIDIR
lightbulb INFERIDA SUGERENCIA

Si una preferencia se detecta en conversación sin que el usuario diga "recuerda", se marca como sugerencia.

CONVERSACIÓN
"...mejor programemos la reunión tipo 16:30 o 17:00, soy más productivo entonces..."
ARCHIVIST · SUGERENCIA
Parece que prefieres reuniones después de las 16:00.
¿Quieres guardarlo como preferencia?
Guardar No guardar No volver a sugerir
Si usuario elige No volver a sugerir, se respeta. No se insiste.
warning CONTRADICCIÓN CONFLICTO DETECTADO

Si existe una memoria anterior que podría contradecir la nueva, se muestra explícitamente.

ESCENARIO
Usuario: "Guarda que Daniel prefiere reuniones por la mañana."
SISTEMA DETECTA
M-201: Daniel prefiere reuniones después de las 16:00 (15 Jul)
UI MUESTRA
Encontré una preferencia anterior que podría contradecir esta.
Conservar ambas con fechas Reemplazar anterior Descartar nueva
Las contradicciones se muestran PRIMERO, no se ocultan en el resumen.
CAP. 06 · INFERIDA VNBOT APP FLOW 32 / 57
  • Inferida: preferencia detectada sin "recuerda" se ofrece como sugerencia.
  • 3 opciones: guardar, no guardar, no volver a sugerir (se respeta).
  • Contradicción: memoria anterior conflictiva se muestra antes de guardar la nueva.
  • 3 opciones: conservar ambas con fechas, reemplazar anterior, descartar nueva.
  • Principio: contradicciones se muestran PRIMERO, no se ocultan.
