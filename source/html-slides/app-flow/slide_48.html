Offline — Permitidas vs No Ejecutadas

check_circle
PERMITIDAS
Funcionan completamente offline.

Operaciones que funcionan completamente offline. Se encolan para sync si modifican datos.

check Crear memoria local
check Editar memoria local
check Crear recordatorio local
check Completar recordatorio local
check Consultar datos cacheados
check Grabar audio (si dispositivo lo permite)

schedule Scheduler local mantiene recordatorios funcionando sin red.

⇄
PARTICIÓN
cancel
NO EJECUTADAS
Requieren conexión.

Operaciones que requieren conexión. Se rechazan o encolan según tipo.

close Enviar email
close Consultar MCP remoto
close Crear evento externo
close Sincronizar con calendario
close Generar respuesta mediante LLM cloud
MSG AL USUARIO
"Requiere conexión"

priority_high Estas operaciones muestran mensaje claro: "Requiere conexión". NO se encolan silenciosamente.
