CAP. 01 · CICLO 05 / 57
- El ciclo central de VNBOT tiene 7 etapas conectadas linealmente. - La etapa CONFIRMAR (4) es condicional — solo si hay riesgo. - 3 roles separados: IA propone, Dominio valida, Backend audita. - La separación de roles evita que el LLM decida directamente acciones. - El usuario siempre puede auditar qué se entendió y qué se ejecutó.
