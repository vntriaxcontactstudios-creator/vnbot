08 // PLATAFORMAS

Web/PWA · Desktop · APK · CLI

Mismo lenguaje visual, distintas restricciones por runtime.

public WEB / PWA BASE
Responsive 360-1920px. IndexedDB offline. Service worker cache. Instalable como PWA.
Breakpoints: 360 / 768 / 1280 / 1920
Storage: IndexedDB
Offline: service worker + cache strategies
Distribución: GitHub Pages
desktop_windows DESKTOP NATIVE
Windows, Linux, macOS. Ventana nativa con marco HUD. Atajos de teclado globales. SQLite local.
Framework: Tauri o Electron
Storage: SQLite local
Atajos: globales + locales
Distribución: GitHub Releases
phone_android APK ANDROID TOUCH
Touch-first. Gestos nativos. Notificaciones push. Share intent. Mascota en widget de home.
Min SDK: 26 (Android 8+)
Storage: Room (SQLite)
Notif: FCM o self-hosted
Distribución: GitHub Releases + F-Droid
terminal CLI TERMINAL
Output formateado ANSI. Mascota como ASCII art opcional. Subcomandos: capture, search, agents, sync. TTY y pipe-friendly.
Binario: estático, sin runtime
Output: ANSI / JSON / YAML
Mascota: ASCII art (--mascot)
Distribución: npm + brew + cargo
