- Documento Maestro.
- PRD.
- TRD.
- Esquema Backend.
- Plan de Implementación.
- Diseño UI/UX.
- App Flow.
- Modelo de Datos.
- Seguridad y Privacidad.
- MCP y Skills.
- Roadmap.
- Estrategia de Sincronización (nuevo).
- Testing y Observabilidad (nuevo).
- Gobernanza de Proyecto (nuevo).
- README de documentación.

## Referencias visuales

La carpeta `visual-references/` contiene las imágenes adjuntas y renombradas para facilitar su identificación.

El análisis de dichas referencias está en:

```text
VISUAL_REFERENCES_AND_ANALYSIS.md
```

Las imágenes con marcas de agua se incluyen únicamente como material de referencia y análisis; no son assets de producción.

## Assets generados

La carpeta `assets-generated/` contiene prototipos originales de VNBOT:

- `vnbot-golem-mascot-variants.png` — 6 variantes conceptuales del golem.
- `vnbot-golem-animation-sheet.png` — hoja conceptual de estados (original).
- `vnbot-golem-agent-spritesheet.png` — 7 agentes diferenciados con accesorios y paletas.
- `vnbot-golem-state-spritesheet.png` — 10 estados del golem principal.
- `vnbot-golem-ui-emotes.png` — 12 emotes compactos para chat, toasts y badges.

## Referencia técnica de sprites
